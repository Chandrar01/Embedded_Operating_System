
to run the python file OPEN TERMINAL from the file location and issue 
following commands as mentioned below


for RANDOM page replacement algorithm 
	python3 PA4_RAND.py

for FIFO page replacement algorithm 
	python3 PA4_FIFO.py
result will be displayed in CONSOLE output window

for LRU page replacement algorithm 
	python3 PA4_LRU.py
result will be displayed in CONSOLE output window
